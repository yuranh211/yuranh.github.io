## 002 公式一：能量

1. 智商很难后天提高，情商却可以得到有效训练


2. 定义情商：高情商，本质上是**高能量**。首先你要<span style="font-size: 18px; color: red">满足自己</span>的需求，其次再来<span style="font-size: 18px; color: red">成全别人</span>，最后达成双赢的局面


3. “00后整顿职场”，最后把公司整垮了，这肯定不是高情商的表现。与之相反的是，《老板管理手册》


4. <mark style="font-size: 18px; background-color: #fff88f; color: black">AEIOU 法</mark>，可以用这个方法来监控我们的能量

| **A** |  activity   | **活动** | 结构性的？非结构性的？ |

|   E   | environment |   环境   |   室内？户外？    |
|   I   | interaction |   互动   |  熟悉的？陌生的？   |

## 003 公式二：特点

1. 认识你自己，是建立能量场的第一步

2. 我们不赞成用优点或者缺点的说法，因为那都是你的特点

3. <span style="font-size: 18px; color: red">喜恶同因</span>：喜欢你的人和厌恶你的人，可能出于同一个原因

4. 同样地，你同一个特点，放在**不同的应用场景**，就会导致优势或者劣势

5. 与其想怎么把缺点改正，不如想怎么把这个特点用在对的地方

6. <mark style="font-size: 18px; background-color: #fff88f; color: black">我们要做的是，找到自己的特点，并且用好自己的特点</mark>

### （1）如何找到自己的特点？

方法一：把自己之前做过的、比较有记忆度的事情列出来（成功的、糟糕的都要有），然后分析事情背后体现出来的自己的特点

方法二：套理论模型。包括：MBTI 人格测试；星座特质；大五人格模型；九分仪模型等


> 所有模型都是错的，但所有模型又都是有用的

### （2）如何利用自己的特点？

1.彼得 • 德鲁克 的【回馈分析法】

2.在面临重大选择或者做出关键行动之前，**事前**自己预想一下结果，最后拿**事后**结果与之比较分析

3.根据比较结果，调整自己的动作，以及转换应用场景


4.一次正确的预测，要基本符合 SMART 原则

## 004 公式三：自卑

1.你不是因为自卑才不去做，而是<span style="font-size: 18px; color: red">你需要自卑这个借口</span>，来掩饰自己的不想做。<mark style="font-size: 18px; background-color: #fff88f; color: black">自卑是一个人的主动选择</mark>

2.很多人的状态是“躺又躺不平，卷又卷不赢”。如果真正地躺平，不在乎胜负、输赢、名利，倒也过得轻松自在。所谓“朝臣待漏五更寒，铁甲将军夜渡关。山寺日高僧未起，算来名利不如闲。”可是，做这样的“废物”，需要极高的天赋和悟性。

3.既如此，就不要再拿自卑当做借口，而是转化成驱动力

4.很多人感到自卑，其实是太在乎外界对你的评价。别人<span style="font-size: 18px; color: red">（包括亲密关系和陌生人）</span>否定你，或者对你的行为没有正向反馈，都不必在意。<mark style="font-size: 18px; background-color: #fff88f; color: black">因为你不需要来自别人的评价。</mark>  

5.真正值得关注的是：自己对自己真实的看法；结果给自己带来的反馈

6.怎么才能通过结果的反馈给自己带来自信呢？老王教的 “化生为熟、化繁为简”，<span style="font-size: 18px; color: red">从简单入手！</span> 即：做自己擅长的事情；或者将目标拆解成一个个容易完成的小行动，走一步再走一步

7.【杨天真】<span style="font-size: 18px; color: red">人会变得自信，是你在正确的方向上做着不断成功的事情。你知道自己做的是对的，且结果也不断验证你是对的</span>

## 005 公式四：自律 & 习惯

谈自律

1.我上来先说一个暴论：<span style="font-size: 18px; color: red">想自律，先放弃</span>。我们的人生，只需要关注自己<mark style="font-size: 18px; background-color: #fff88f; color: black">一定要的</mark>和自己<mark style="font-size: 18px; background-color: #fff88f; color: black">绝对不想失去的</mark>，其他的都不重要

2.那些无关紧要、不痛不痒的事情，不在我们分配时间和精力的范畴，更谈不上自律了

3.当你知道自己是可以放弃的时候，你的内心是自由的

4.比如刷手机这件事，一天刷 10 个小时的手机、一天刷 5 个小时的手机 和 不刷手机，对我人生的本质、对我想要的东西有影响吗？没有，所以它不重要。我想控制就控制，不想控制就不控制

5.结果导向，聚焦于我们的目标，再谈自律

6.习惯是最好的自律，因为<span style="font-size: 18px; color: red">它不需要决策成本</span>。你想要坚持做一件事，最简单的方法就是养成习惯。不需要动用意志力，不需要思考，也就没有所谓的内耗

---

习惯习惯，习得的惯性。它可以分为两个方面：一是做有利的事情；二是不做有害的事情

针对第一个方面：**微习惯 = 锚点时刻 + 微行为 + 庆祝 （正向反馈）**。这个模型的巧妙之处在于：把我们已经在做的行为当成提示，促进我们培养新的习惯。这里我不再过多赘述。

针对第二个方面，我浅谈一下我的看法（可能说的不太全面、准确，因为这个话题挺大的）

（1）<span style="font-size: 18px; color: red">迎难则绕</span>。物理隔离，增大行为发生的摩擦力（成本）。比如：平时散步时，碰到奶茶店，总控制不住奖励自己。那么以后刻意避免有奶茶店的散步路线。这也是“孟母三迁”的原理

（2）<span style="font-size: 18px; color: red">寻找替代方案</span>。还是上面那个例子，你可以给自己点一杯酸奶，而不是含糖量高的奶茶🧋。日常生活中，我们需要寻找自己热爱的东西，来替换不良嗜好。

（3）<span style="font-size: 18px; color: red">保持记录</span>。你不需要为了美化数据而造假，只需要客观、真实地记录就好

（4）以我为例，我特别容易吃多。至于原因嘛，可以用马斯洛需求层次来解释。最底层的生理需求满足之后，由于长期缺乏，心理层面的安全感还没有满足。我想到的方法是：我不囤积令我狂吃的零食（豆干、辣条、可乐等），取而代之是一些全麦面包和甜点。

（5）再如我会沉溺于短视频流无法自拔，这时候增大行为发生的摩擦力。具体方法有：手机直接关机；把手机放到不容易取得的地方；<mark style="font-size: 18px; background-color: #fff88f; color: black">关闭手机音量，也就是看无声视频</mark>（这个方法蛮好用的😅）

（6）**贪多嚼不烂**。你一次只需要关注一两个习惯，持续记录一两个习惯的每一天就好。不要给自己设立半年、一年、乃至更长远的flag

（7）可能有人质疑这样做的意义：多麻烦啊，我相信我自己。。。呵，我只想说不要考验人性。<span style="font-size: 18px; color: red">你还没有达到百毒不侵的境界</span>。我们做任何事情都要尊重人性，这是最大的规律。我们养成习惯也是为了让自己更快乐、更幸福，而不是相反，所以要减少过程中的痛苦。


## 006 公式五：失败

你先告诉我，你怎么定义你的失败，衡量失败的标准有哪些？然后你再往下看

失败只是当前某个事情的结果。我们如何对待失败，才影响后面事情的发展

正确面对失败的三部曲：

### （1）接受失败

成功是偶然的，不仅需要集齐所有要素，还要有极大的运气。从这个角度说，失败是必然的

任何成功都是阶段性的，失败也是。<span style="font-size: 18px; color: red">把时间这个维度拉长</span>，结果自然烟消云散


### （2）调整情绪

不以物喜，不以己悲

有情绪很正常。但你迟迟走不出情绪就不对

<mark style="font-size: 18px; background-color: #fff88f; color: black">负面情绪是失败的后遗症</mark>

当你真正接受失败这个结果，并且走出自己的情绪，这件事情才算收尾


### （3）有效复盘

在我们身上发生的事情，不能让它白白过去！通过有效复盘，让经历变成经验

| **第一步** |      总结事实，发生了什么？      |
| :-----: | :-------------------: |
|   第二步   |  判断哪些地方做的好，哪些地方做的不好？  |
|   第三步   |    如何调整？把不够好的地方做好     |
|   第四步   | 还有哪些？我根本没有考虑到，以后可以补充上 |

分享一个我在罗胖2024年跨年演讲时听到的观点：

未来是什么？未来是我们当下某个因素的展开。从这个意义上说，当下即是未来。

大胆地去做事，勇敢地将世界占为己有。至于事情的结果，曼德拉早已给出了答案：<span style="font-size: 18px; color: red">我从未失败过。要么赢得胜利，要么学到东西</span>
